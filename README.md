## Frog Simulation Backend

This is a simulation based Frog Asia challenge. It has 3 main API endpoints

- [Register](/api/register).
- [Login](/api/login).
- [Create Simulation](/api/simulation) 

After you register and login, using the token you can create a simulation

## Installation

git clone 
composer install
npm install
php artisan migrate
